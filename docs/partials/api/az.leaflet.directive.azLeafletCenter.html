azLeafletCenter
directive in module az.leaflet

Description

Helper directive to two-way bind the leaflet map center/zoom, must be nested within an azLeaflet directive

Dependencies

  • azLeaflet

Usage

as element:
<az-leaflet-center
       center="{L.LatLng}"
       zoom="{Integer}"
       [options="{L.ZoomPanOptions}"]>
</az-leaflet-center>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
centerL.LatLng
  • Center of map, watched for changes
zoomInteger
  • Zoom level of map, watched for changes
options
(optional)
L.ZoomPanOptions
  • Zoom/Pan options object to use to control the zoom/pan
