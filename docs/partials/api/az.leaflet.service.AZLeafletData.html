AZLeafletData
service in module az.leaflet

Description

Handles storage and retrieval of L.map objects instantiated via azLeaflet directive

Methods

  • deleteMap(scopeId)

    Remove the map with id from the AZLeafletData cache.

    DOES NOT remove the map from the DOM or do any other associated cleanup tasks

    Parameters
    ParamTypeDetails
    scopeId
    (optional)
    		String
    • Id of map to delete from AZLeafletData. Optional if there is only one initialized map.

  • getMap(scopeId)

    Get map with the given id. If only one map is saved to AZLeafletData, the id can be omitted and getMap will return the only map object it is caching.

    Parameters
    ParamTypeDetails
    scopeId
    (optional)
    		String
    • Id of map to retrieve from AZLeafletData. Optional if there is only one initialized map.
    Returns
    Promise

    Resolves with the requested L.map object

  • setMap(map, scopeId)

    Save map to AZLeafletData with the given id

    Parameters
    ParamTypeDetails
    mapL.map
    • Map object to save
    scopeIdString
    • Unique string key to save the map with, use this id to retrieve later with getMap()
