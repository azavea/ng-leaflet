AZLeafletDefaultsProvider
object in module az.leaflet

Description

AZLeafletDefaults provides the default L.MapOptions object for each new map that is instantiated via the directive

Methods

  • setDefaults(newDefaults)

    Extend the default AZLeafletDefaults object

    Default object is:

    var defaults = {
    center: [0,0],
    zoom: 1,
    crs: L.CRS.EPSG3857
};
