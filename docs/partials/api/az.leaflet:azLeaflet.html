az:leaflet:azLeaflet

Description

Instantiate a new leaflet map on the element, with options, which are not watched for changes

A class azavea-ng-leaflet-map is added to the root Leaflet map element

This directive does not handle addition of any basemaps, etc. This must all be done in child directives

Events

az.leaflet.invalidateSize: Triggers a L.map.invalidateSize() call on next digest cycle

Usage

as element:
<az-leaflet
       width="{String}"
       height="{String}"
       options="{Object}">
</az-leaflet>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
widthString
  • (Optional) Width of map in px or percentage
heightString
  • (Optional) Height of map in px or percentage
optionsObject
  • (Optional) Default map options to instantiate map with. If not provided, az.leaflet:AZLeafletDefaults is used.
